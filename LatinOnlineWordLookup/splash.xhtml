LatinOWL — Latin Online Word Lookup

The data that LatinOWL uses comes from the Perseus Digital Library, and remains entirely theirs. It is used under their open source principles.

Please note that loading the data requires an active internet connection.

Many thanks must go to Matt Gallagher of CocoaWithLove for the 'XPathQuery' class used in this project. I am very glad that he built that wrapper around libxml2and not me. This project also uses the 'Reachability' class from Apple. Those source files retain their copyright notices in the source repository.
